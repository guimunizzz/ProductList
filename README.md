# Lista de Compras com React

## Descricao da atividade

Desenvolver uma aplicacao de Lista de Compras utilizando React, permitindo ao usuario adicionar e remover itens dinamicamente, com atualizacao automatica da interface sem recarregar a pagina.

Os itens devem ser renderizados com base no estado da aplicacao, utilizando `useState`.

## Requisitos da aplicacao

A interface deve conter:

- Campo para informar o nome do produto
- Campo para informar a quantidade
- Botao para adicionar o item a lista
- Lista exibida na tela

Cada item da lista deve exibir:

- Nome do produto
- Quantidade
- Botao para remocao

## Regras de funcionamento

A aplicacao deve:

- Permitir remover itens individualmente
- Limpar os campos apos a adicao
- Atualizar a interface automaticamente ao adicionar ou remover itens

## Objetivo tecnico

Esta atividade tem como foco praticar:

- Manipulacao de estado com `useState`
- Renderizacao dinamica de listas no React
- Atualizacao reativa da interface sem recarregar a pagina
- Interacao com formularios e eventos

## Execucao do projeto

Para iniciar o projeto em ambiente de desenvolvimento:

```bash
npm install
npm run dev
```

Para gerar a versao de producao:

```bash
npm run build
```
